Operating Systems

This repository contains notes, assignments, and code from the Operating Systems course at **FAST NUCES**.

📖 About This Course

Operating Systems explores how software manages hardware resources and provides services to programs. The course covers the core components of modern operating systems through theory and practical examples.

🗂️ Topics Covered

- Process management and scheduling algorithms
- Threads and concurrency
- CPU scheduling (FCFS, SJF, Round Robin, Priority)
- Memory management and virtual memory
- Paging and segmentation
- Deadlocks — detection, prevention, and avoidance
- File systems and I/O management
